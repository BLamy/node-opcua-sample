node-opcua-sample
=================

a simple OPCUA sample client to demonstrate how to use the [node-opcua](https://github.com/erossignon/node-opcua) SDK.



    $ git clone https://github.com/erossignon/node-opcua-sample
    $ npm install
    $ node simple_client.js
